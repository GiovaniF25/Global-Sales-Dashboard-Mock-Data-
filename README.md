Simple Sales Dashboard (Mock Data)
This is a basic portfolio project designed to showcase Power BI development integrated with Git version control.

🖥️ Report Features
A straightforward two-page report using fictional data:

General Analysis: High-level Sales and Revenue metrics.

Continental Analysis: Simple profit comparison across global regions.

🛠️ Technical Focus
Format: Built using the .pbip project extension for granular tracking.

Version Control: Uses Branches (like refactor/) to manage small updates and naming conventions independently from the main file.
